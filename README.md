# FinExtract AI - Backend

AI-powered Financial Statement Extraction API.

## Features
- PDF Upload
- Multi-year extraction
- Structured JSON via LLM
- CSV Output

## Tech Stack
- FastAPI
- OpenAI GPT-4o-mini
- pdfplumber
- Pandas

## Run Locally

pip install -r requirements.txt  
uvicorn main:app --reload

## Deployment
Designed for Render free tier.
